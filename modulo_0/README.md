# Modulo 0: Nivelacion

> Prerequisitos fundamentales para aprovechar el curso de TOGAF Standard.

---

## Descripcion

Este modulo proporciona los conocimientos base necesarios antes de iniciar el estudio de TOGAF. Cubre conceptos esenciales de TI empresarial, gestion de proyectos, modelado y estrategia de negocio que se utilizan a lo largo del curso.

**Duracion**: 3 horas
**Nivel**: Basico
**Prerrequisitos**: Ninguno

---

## Objetivos de Aprendizaje

Al completar este modulo, seras capaz de:

- Explicar los componentes principales de TI empresarial
- Entender el ciclo de vida de proyectos y metodologias
- Leer y crear diagramas basicos (UML, BPMN)
- Comprender conceptos estrategicos y su relacion con TI
