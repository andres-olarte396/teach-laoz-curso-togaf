# Modulo 3: Arquitectura de Negocio

> Phase B del ADM: Definiendo las capacidades y procesos del negocio.

---

## Descripcion

Este modulo cubre la Phase B del ADM, enfocada en desarrollar la arquitectura de negocio. AprenderÃ¡s a modelar capacidades, procesos y estructuras organizacionales, asi como a realizar gap analysis y aplicar la tecnica de Business Scenarios.

**Duracion**: 6 horas
**Nivel**: Intermedio
**Prerrequisitos**: Modulo 2 (ADM Ciclo Completo)

---

## Objetivos de Aprendizaje

Al completar este modulo, seras capaz de:

- Crear modelos de capacidades de negocio
- Documentar procesos de negocio clave
- Mapear estructuras organizacionales y roles
- Realizar gap analysis entre baseline y target
- Aplicar la tecnica de Business Scenarios
