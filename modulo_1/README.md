# Modulo 1: Fundamentos de TOGAF

> Introduccion al estandar TOGAF y sus conceptos fundamentales.

---

## Descripcion

Este modulo presenta los fundamentos del estandar TOGAF, su historia, estructura y terminologia esencial. Establece la base conceptual necesaria para comprender el resto del framework.

**Duracion**: 4 horas
**Nivel**: Introductorio
**Prerrequisitos**: Modulo 0 (Nivelacion) o conocimientos equivalentes

---

## Objetivos de Aprendizaje

Al completar este modulo, seras capaz de:

- Explicar la historia y evolucion de TOGAF
- Identificar los cuatro pilares del estandar
- Utilizar correctamente la terminologia de TOGAF
- Diferenciar arquitectura de diseno
- Entender baseline vs target architecture
- Aplicar el concepto de viewpoints y views

## Conceptos Clave

### Los Cuatro Pilares de TOGAF

```
+------------------+------------------+
|       ADM        |    Content       |
| (Metodo)         | Framework        |
|                  | (Que producir)   |
+------------------+------------------+
|   Enterprise     |   Capability     |
|   Continuum      |   Framework      |
|   (Repositorio)  |   (Como operar)  |
+------------------+------------------+
```

### Dominios de Arquitectura

| Dominio | Enfoque |
|---------|---------|
| Business | Capacidades, procesos, organizacion |
| Data | Entidades, flujos, gobernanza |
| Application | Sistemas, integraciones, APIs |
| Technology | Infraestructura, plataformas, cloud |
