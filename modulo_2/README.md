# Modulo 2: ADM - Ciclo Completo

> El corazon de TOGAF: Architecture Development Method.

---

## Descripcion

Este modulo cubre el ADM (Architecture Development Method), el proceso central de TOGAF para desarrollar arquitectura empresarial. Incluye la vision general del ciclo, las fases Preliminary y A, y como adaptar el ADM a diferentes contextos.

**Duracion**: 8 horas
**Nivel**: Intermedio
**Prerrequisitos**: Modulo 1 (Fundamentos TOGAF)

---

## Objetivos de Aprendizaje

Al completar este modulo, seras capaz de:

- Explicar la estructura completa del ciclo ADM
- Identificar inputs, outputs y steps de cada fase
- Adaptar el ADM a diferentes escenarios
- Ejecutar la fase Preliminary correctamente
- Desarrollar una Architecture Vision (Phase A)
- Gestionar stakeholders efectivamente

## Ejercicio Practico

Crea un Architecture Vision para un proyecto en tu organizacion:

1. Define el Request for Architecture Work
2. Identifica los stakeholders clave
3. Documenta la vision de alto nivel
4. Establece los principios que guiaran el proyecto
