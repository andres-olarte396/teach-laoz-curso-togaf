# Modulo 10: Proyecto Integrador Final

> Aplicando todo lo aprendido en un caso real.

---

## Descripcion

Este modulo presenta un proyecto integrador completo basado en el caso RetailCorp, una cadena de tiendas en proceso de transformacion digital. Los participantes aplicaran el ciclo ADM completo para desarrollar una arquitectura empresarial profesional.

**Duracion**: 6 horas
**Nivel**: Avanzado
**Prerrequisitos**: Modulos 1-9

---

## Objetivos de Aprendizaje

Al completar este modulo, seras capaz de:

- Analizar un escenario de transformacion empresarial
- Aplicar el ciclo ADM de principio a fin
- Desarrollar arquitectura en los 4 dominios
- Crear gap analysis y roadmaps realistas
- Presentar y defender decisiones arquitectonicas

## El Caso: RetailCorp

### Perfil de la Empresa

| Aspecto | Detalle |
|---------|---------|
| Industria | Retail (tiendas departamentales) |
| Ingresos | $2.5 billones anuales |
| Empleados | 15,000 |
| Tiendas | 85 tiendas fisicas |
| Ecommerce | 8% de ventas (objetivo: 35%) |

### Desafio de Transformacion

```
SITUACION ACTUAL              SITUACION OBJETIVO
================              ==================
Canales separados     -->     Omnicanalidad
Sin Customer 360      -->     Personalizacion
Sistemas legacy       -->     Cloud-native
6+ meses TTM          -->     < 4 semanas TTM
```

### Alcance del Proyecto

- Fases ADM: Preliminary hasta Phase F
- Dominios: Business, Data, Application, Technology
- Horizonte: 2026-2028 (3 anos)
- Inversion: $25M
