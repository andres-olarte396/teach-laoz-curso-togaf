# TOGAF® Standard - Curso de Arquitectura Empresarial

> Curso completo de 61 horas para dominar el framework de arquitectura empresarial mas utilizado en el mundo.

---

## Descripcion

Este curso proporciona una formacion integral en **TOGAF® Standard**, el marco de referencia global para arquitectura empresarial desarrollado por The Open Group. A traves de 11 modulos estructurados, los participantes aprenderan a disenar, planificar e implementar arquitecturas empresariales que alineen la tecnologia con los objetivos de negocio.

### Objetivos de Aprendizaje

Al completar este curso, seras capaz de:

- Comprender la estructura y componentes del estandar TOGAF
- Aplicar el ciclo ADM (Architecture Development Method) completo
- Disenar arquitecturas en los 4 dominios: Negocio, Datos, Aplicaciones y Tecnologia
- Crear roadmaps de transformacion y arquitecturas de transicion
- Implementar gobierno de arquitectura efectivo
- Gestionar el repositorio y building blocks empresariales
- Evaluar y mejorar la madurez de la practica de arquitectura
