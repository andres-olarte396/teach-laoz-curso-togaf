# Modulo 9: Marco de Capacidad de Arquitectura

> Construyendo y madurando la practica de arquitectura empresarial.

---

## Descripcion

Este modulo cubre el Architecture Capability Framework de TOGAF, incluyendo los roles del equipo de arquitectura, el desarrollo de skills y el modelo de madurez (ACMM). Estos elementos son esenciales para establecer una practica de EA sostenible.

**Duracion**: 4 horas
**Nivel**: Avanzado
**Prerrequisitos**: Modulo 8 (Building Blocks y Repositorio)

---

## Objetivos de Aprendizaje

Al completar este modulo, seras capaz de:

- Definir los roles del equipo de arquitectura
- Identificar skills necesarios para arquitectos
- Disenar planes de desarrollo profesional
- Evaluar madurez de la practica EA con ACMM
- Crear roadmaps de mejora de capacidad


### Roles Principales

| Rol | Foco | Alcance |
|-----|------|---------|
| Chief Architect | Estrategia | Organizacion |
| Enterprise Architect | Coherencia | Enterprise |
| Domain Architect | Especialidad | Dominio |
| Solution Architect | Diseno | Proyecto |

---

## Modelo de Madurez ACMM

```
Nivel 5: OPTIMIZING    - Mejora continua, innovacion
Nivel 4: MANAGED       - Metricas, valor demostrado
Nivel 3: DEFINED       - Procesos estandar, governance
Nivel 2: DEVELOPING    - Procesos basicos, equipo
Nivel 1: INITIAL       - Ad-hoc, heroes individuales
Nivel 0: NONE          - Sin arquitectura
```

## Ejercicio Practico

1. Define los roles de arquitectura para tu organizacion
2. Crea una matriz de skills requeridos vs disponibles
3. Realiza un assessment de madurez ACMM
4. Desarrolla un roadmap para subir 1 nivel de madurez
